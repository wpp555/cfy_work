3、与bootstrap的模态框(modal)组合产生的问题

html代码：

				
					<div id="myModal" class="modal fade" tabindex="-1" role="dialog" aria-hidden="true">
					<select id="mySelect2">
					<option value="3">baidu</option>
					<option value="4">sina</option>
					<option value="5">huawei</option>
					</select>

js代码：

				
					$("mySelect2").select2({
					dropdownParent: $('#myModal')
					});

5、select选择事件

js代码：

				
				    var select = $(".js-example-select").select2({
						width: "resolve"
					});
					select.on('select2:select', function (e) {
						var data = e.params.data; //获取当前点击的option对象，里面包含value、text等
						console.log(data);
					});
